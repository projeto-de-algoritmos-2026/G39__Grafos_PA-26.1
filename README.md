## Outros 
Trabalho 1: consiste numa aplicação de linha de comando (CLI) em Java. O sistema gera um labirinto interativo modelado como um grafo em grade (grid), cujo objetivo é calcular a rota mais eficiente para a recolha de moedas utilizando o **Algoritmo de Dijkstra**.

O algoritmo avalia e escolhe as rotas minimizando o custo total, tomando decisões matemáticas para desviar ou atravessar obstáculos com pesos diferentes.
